\chapter{Introduction}
The Stationery Management System is a graphical desktop application designed using Python's built-in \texttt{tkinter} library. It provides a clean, modular, and user-friendly interface to record item quantities, compute total prices, and generate a formatted receipt. The system aims to simplify retail-level stationery billing, automate manual calculations, and ensure error-free record keeping.

\section*{Objectives}
\begin{itemize}[leftmargin=*]
    \item To develop a functional GUI using Python's standard libraries.
    \item To implement modular, maintainable, and extensible program structure.
    \item To calculate and display total stationery cost dynamically.
    \item To provide a receipt-style output for clarity and practical use.
\end{itemize}

\section*{Scope of the Project}
This software targets small stationery stores, school inventory systems, and quick estimation needs. It operates entirely offline, requires no external dependencies, and provides a minimal yet highly functional workflow.

% ==========================
% Chapter 2 — System Design
% ==========================
\chapter{System Design}

\section*{Overall Architecture}
The application follows a clean separation of concerns:\\
\textbf{Interface Layer} handles layout, input fields, and interactive elements.\\
\textbf{Logic Layer} computes totals, validates input, and generates receipts.\\
\textbf{Data Layer} stores item prices and maintains quantity variables.

\section*{Data Structures}
Prices are stored in a Python dictionary of the form:
\begin{verbatim}
ITEMS = {
    "Pencil": 5,
    "Eraser": 3,
    ...
}
\end{verbatim}
Quantities are managed using \texttt{IntVar()} for live updates and validation.

\chapter{Implementation}

\section*{Programming Language}
The project is written entirely in Python 3.x using the standard \texttt{tkinter} library.

\section*{Key Features}
\begin{itemize}[leftmargin=*]
    \item Real-time total computation.
    \item Automatic sanitization of invalid entries.
    \item Clean, structured UI layout.
    \item Auto-generated receipt with itemized details.
\end{itemize}

\section*{Execution}
Run the application using:\\
\texttt{python3 "STATIONARY MANAGNMENT final.py"}
